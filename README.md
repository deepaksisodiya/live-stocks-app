# Live Stocks App

## Technologies

* [React](https://facebook.github.io/react/index.html) - A Javascript Library For Building User Interfaces

## How to setup

```
git clone https://github.com/deepaksisodiya/live-stocks-app.git
npm install
```

## How to run the app

```
npm start
```
